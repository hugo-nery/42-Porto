*This project has been created as part of the 42 curriculum by hde-albu.*

-Description:
	The idea of this project is to build a set of functions (my own library), so I can use it for future programs during the Common Core. 

-Instructions:
	This project includes a Makefile, therefore (in the terminal):
		-'make' will compile all the functions/header to the file libft.a;
		-'clean' will remove the generated .o files;
		-'fclean' will run 'clean' and also remove the file libft.a;
		-'re' will will delete everything and re-compile a new libft.a;

-Resources:
